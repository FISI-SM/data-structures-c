# Cola usando arreglo dinámico (buffer circular) en C++

Implementa `ColaArray<T>` con `new[]/delete[]`, crecimiento automático y **índices circulares**.

## API
- `push(const T&)`, `pop()`
- `front()`, `back()`
- `empty()`, `size()`, `capacity()`, `clear()`, `reservar(size_t)`
- `imprimir()`

## Compilar y ejecutar
```bash
make
./bin/app_cola_array
```
