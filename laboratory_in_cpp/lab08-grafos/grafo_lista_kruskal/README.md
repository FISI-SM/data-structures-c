# Grafo con Lista de Adyacencia (lista enlazada) en C++

Implementa `GrafoLista` ponderado con un arreglo de punteros a listas (`NodoAdj`). Operaciones: `agregarArista`, `eliminarArista`, `existeArista`, `imprimir`, `kruskal`.

El algoritmo de Kruskal encuentra el árbol de expansión mínima (MST) de un grafo no dirigido ponderado, seleccionando las aristas de menor peso que no formen ciclos.

## Compilar y ejecutar
```bash
make
./bin/app_grafo_lista
```
